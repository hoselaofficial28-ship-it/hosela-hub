Upload Absensi

Data fingerprint mingguan

Cara Upload Data
1
Export file Excel dari mesin fingerprint seperti biasa
2
Buka file Excel di komputer
3
Pilih semua (Ctrl+A) → Salin (Ctrl+C)
4
Buka Google Sheets Hosela Hub — Database → tab ABSENSI_UPLOAD
5
Klik cell A1 → Tempel (Ctrl+V)
6
Kembali ke sini dan klik Proses Data di bawah
⚡ Sistem otomatis menghitung kehadiran, keterlambatan, absen, dan lembur. Sheet ABSENSI_UPLOAD akan dikosongkan setelah selesai.
